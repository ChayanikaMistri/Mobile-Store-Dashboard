# 📊 Mobile Store Sales Dashboard

This project is an interactive **Power BI dashboard** built to analyze mobile store sales data over a one-year period across India. The dashboard provides insights into sales performance, customer behavior, and product trends across multiple dimensions.

---

## 🔍 Dashboard Features

- 📅 **Monthly Sales Trend Analysis** → Identify growth patterns and seasonal spikes  
- 🗺️ **State-wise Sales Visualization (India)** → Regional performance comparison  
- 📱 **Sales by Mobile Model** → Product-level insights  
- 💳 **Payment Method Breakdown** → Customer transaction preferences  
- 🏷️ **Brand-wise Revenue Analysis** → Competitive positioning  
- ⭐ **Customer Ratings Analysis** → Quality vs. sales relationship
